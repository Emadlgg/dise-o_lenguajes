# Diseño de Lenguajes de Programación – CC3071

Este repositorio contiene las **actividades, prácticas y desarrollos** realizados durante el curso **Diseño de Lenguajes de Programación (CC3071)** de la **Universidad del Valle de Guatemala**, correspondiente al **Ciclo 1, 2026**.

El curso se enfoca en el estudio e implementación de las **primeras fases de un compilador**, principalmente el **análisis léxico** y el **análisis sintáctico**, utilizando fundamentos teóricos como **autómatas finitos**, **expresiones regulares** y **gramáticas libres de contexto**.

## Contenido del repositorio

* Actividades prácticas de **análisis léxico**.
* Construcción y minimización de **autómatas finitos (DFA)**.
* Identificación de **lexemas, tokens y atributos léxicos**.
* Diseño de estructuras para **scanners (lexers)**.
* Análisis de **errores léxicos** y estrategias de recuperación.
* Documentación en **PDF** y material de apoyo.
* Videos explicativos (cuando aplica).

## Organización

El repositorio está organizado por **carpetas**, donde cada una corresponde a una **actividad, laboratorio o práctica**, e incluye su propio README descriptivo y los archivos necesarios para su comprensión.
